📝 Publier une mise à jour

Dans PowerShell, à la racine du projet :

$token = "VOTRE_MOT_DE_PASSE"
$uri = "https://bg-remover-api-jt6r.onrender.com/api/admin/upload"
Invoke-RestMethod -Uri $uri -Method POST `
  -Form @{ file = (Get-Item "dist\BG Remover.zip"); version = "1.0.0"; notes = "Description" } `
  -Headers @{ Authorization = "Bearer $token" }

Le fichier sera hébergé directement sur le serveur. Les utilisateurs recevront la mise à jour automatiquement.
